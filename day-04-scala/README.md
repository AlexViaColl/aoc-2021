# [Day 4](https://adventofcode.com/2021/day/4) - [Scala](https://www.scala-lang.org/)

## Quickstart
* Install [Scala](https://www.scala-lang.org/)
* Compile `$ scalac day_04.scala`
* Run `java -cp "/usr/share/scala/lib/scala-library.jar:." Main input.txt`
