# [Day 5](https://adventofcode.com/2021/day/5) - [Erlang](https://www.erlang.org/)

## Quickstart
* Install [Erlang](https://www.erlang.org/)
* Compile `$ erlc day_05.erl`
* Run `erl -noshell -s day_05 solve input.txt -s init stop`
